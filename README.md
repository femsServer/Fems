# FEMS 프로젝트

이 프로젝트는 공장 에너지 관리 시스템(FEMS)의 백엔드 및 프론트엔드 소스 코드를 포함합니다.  
Spring Boot와 React를 기반으로 하며 PostgreSQL + TimescaleDB를 사용합니다.

---

## 📦 개발 환경

### ① BackEnd
- **Spring Boot**: 2.7.18
- **JDK**: Amazon Corretto 11
- **Maven**: 3.8.8

- **Zustand**: 5.0.3
- **Node.js**: 22.11.0

### ③ DB
- **PostgreSQL**: 17.4-1
